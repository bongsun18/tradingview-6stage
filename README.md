# 📊 6스테이지 대순환 — 트레이딩뷰 지표 & 자동매매 전략

고지로 강사의 **이동평균선 6스테이지 대순환 투자법**을 기반으로 한 트레이딩뷰 Pine Script 지표 및 자동매매 전략입니다.

## 🎯 6스테이지 대순환

| 스테이지 | 상태 | 이평선 특징 | 액션 |
|---------|------|------------|------|
| 🟣 제1 | 바닥권 | 역배열 → 수렴 | 관망 |
| 🟢 제2 | 상승 초기 | 골든크로스 발생 | **매수 (50%)** |
| 🟩 제3 | 상승 추세 | 정배열 완성 | 보유 |
| 🟠 제4 | 천정권 | 이격 + 다이버전스 | **10% 매도** |
| 🔴 제5 | 하락 초기 | 데드크로스 발생 | **80% 매도** |
| ⬛ 제6 | 하락 추세 | 역배열 완성 | **전량 매도** |

## 📊 지표 (`6stage_indicator.pine`)

### 기능
- **4개 이동평균선** (5/20/60/120일) + 정/역배열 시각화
- **6스테이지 자동 감지** — 배경색 + 라벨로 현재 스테이지 표시
- **다이버전스 감지** — RSI + MACD 기반, 강도 구분 (단일 vs 동시)
- **골든/데드크로스** 마크
- **정보 테이블** — 스테이지, 정/역배열, RSI, MACD, 이격도, 다이버전스, 추세

### 설치
1. 트레이딩뷰 → Pine Editor
2. `6stage_indicator.pine` 내용 복사 & 붙여넣기
3. **Add to Chart** 클릭

## 🔗 POA Bot 연동 (`poa_bot/`)

트레이딩뷰 Alert → Webhook → POA Bot 서버 → 증권사 API 자동 주문

### 아키텍처
```
[트레이딩뷰] → Webhook Alert → [POA Bot 서버] → [한국투자증권 API] → 주문 실행
                                    ↓
                              [Telegram 알림]
```

### 설정 방법

1. **서버 실행:**
```bash
cd poa_bot
cp .env.example .env
# .env에 API 키 입력
pip install -r requirements.txt
uvicorn poa_bot:app --host 0.0.0.0 --port 8000
```

2. **트레이딩뷰 Alert 설정:**
   - 전략 스크립트의 Alert 조건 사용
   - Webhook URL: `http://your-server:8000/webhook`

3. **Webhook JSON 포맷:**
```json
{
  "bot": "6StageBot",
  "action": "BUY|SELL",
  "reason": "STAGE2|BULL_DIV|BEAR_DIV|DEAD_CROSS|FULL_SELL",
  "ticker": "005930",
  "price": 75000,
  "stage": 2,
  "qty_pct": 50
}
```

### API 엔드포인트
| Method | Path | 설명 |
|--------|------|------|
| POST | `/webhook` | 트레이딩뷰 Webhook 수신 |
| GET | `/status` | 봇 상태 조회 |
| GET | `/history` | 주문 이력 조회 |
| GET | `/health` | 헬스체크 |
